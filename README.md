# SmrItems Mod

`SmrItems` — это кастомный мод, разработанный для сервера **SmrProject** на версии Minecraft 1.7.10. Он добавляет в игру множество уникальных предметов, блоков и механик для обогащения игрового процесса.

## Основные возможности

### Вайфу-куклы
В игру добавлены коллекционные декоративные куклы в виде популярных аниме-персонажей. Каждая кукла имеет уникальную модель и текстуру.

### Система кейсов
В игру добавлены кейсы с тематическим лутом. Открывайте их, чтобы получить случайные предметы из популярных модов, а также из ванильного Minecraft.
- **Draconic Case**: Содержит предметы из мода Draconic Evolution.
- **Industrial Case**: Содержит предметы из мода Industrial Craft 2.
- **Thaumcraft Case**: Содержит предметы из мода Thaumcraft.
- **Dakimakura Case**: Содержит предметы из мода Dakimakura.
- **Vanilla Case**: Содержит редкие ванильные предметы.

### Прогрузчики чанков (Якоря мира)
Специальный блок "Якорь мира", который позволяет прогружать чанки, даже когда игрок находится далеко. Для работы требует `Нестабильное топливо` и имеет несколько режимов радиуса прогрузки.

### Прочие особенности
- **Косметические предметы**: Разнообразные аксессуары для кастомизации персонажа, например, ободки для волос.
- **Новые руды**: В мире генерируется `Нестабильная руда`, из которой можно получить ресурсы для крафта других предметов мода.
- **Кастомное главное меню**: Мод заменяет стандартное главное меню Minecraft, добавляя уникальный фон и кнопки для быстрого доступа к серверу SmrProject.
